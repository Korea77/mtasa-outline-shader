# MTA:SA Outline Shader

Exported functions:
```lua
    exports[resourceName]:addOutline(element)
    exports[resourceName]:removeOutline(element)
```

## Overview

This resource applies a shader-based outline effect to any MTA element, creating a highlighted glow around the selected object. The effect uses advanced shader techniques for efficient rendering.
